# BussinessSch

```
## My Task

I will have to create a simple calendar application that allows a user to save events for each hour of the day I will use HTML and CSS and jQuery.
```

```
I used the API [Day.js](https://day.js.org/en/) 
```

```
## User Story


AS a develper i want to create a business scheduale for the busy employees 
I WANT to the employees to be able to add important events to a daily planner
SO THAT they can manage my time effectively
```
```

## Acceptance Criteria


Its done 
WHEN I can open the planner on the web
its done
when the current day is displayed at the top of the calendar
its done
when I scroll down I am presented with time blocks for standard business hours
its done
when I view the time blocks for that day
then each time block is color-coded to indicate whether it is in the past, present, or future
its done
when I click into a time in the future I can enter an event
its done when I click the save button for that time block then its saved in the local storage

its done when I refresh the page then the saved events persist
```
